ATIVIDADE INICIAL PHP

Ao aprender PHP, você entende como criar sites dinâmicos, manipulando dados de formulários, interagindo com bancos de dados (como MySQL), e gerenciando sessões e cookies. 
Aprende a usar estruturas condicionais, loops, funções, e como organizar o código com inclusão de arquivos. Também trabalha com segurança, validando e filtrando dados,
e pode implementar programação orientada a objetos (POO). Além disso, aprende a consumir APIs externas e pode usar frameworks como Laravel para acelerar o desenvolvimento.
